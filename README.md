### 写在开头

1. 本项目全程使用uv进行环境管理，所以建议你也使用uv来。这里是uv的配置方式:[how-to-install-uv](github.com/volcengine/veadk-python?tab=readme-ov-file)
2. veadk所有key的配置都依赖`config.yaml`（其实也可以用环境变量），所以希望你牢牢记得，别把这东西泄露出去。
